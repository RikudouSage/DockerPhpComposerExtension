Docker image that includes php, latest version of composer
and the following extensions:

- bcmath
- bz2
- curl
- gd
- imap
- intl
- json
- mbstring
- mysql
- opcache
- pgsql
- readline
- sqlite3
- xml
- zip
- imagick
- mailparse
- redis

## Supported tags:

- 7.1
- 7.2
- 7.3
- 7.4-beta
- latest - points to latest stable version
